# پروژه داینامیک کانون همیاری

این پروژه یک وبسایت داینامیک با فریم‌ورک Django است که تمام محتوای آن از طریق پنل ادمین قابل مدیریت است.

## ویژگی‌ها

- ✅ **مدیریت کامل محتوا**: تمام محتوا از طریق پنل ادمین قابل ویرایش است
- ✅ **مدل‌های رابط‌ای**: روابط بین جداول به درستی تعریف شده است
- ✅ **پنل ادمین حرفه‌ای**: پنل ادمین با امکانات کامل و قابل تنظیم
- ✅ **فرم‌های داینامیک**: تمام فرم‌ها با Django Forms پیاده‌سازی شده‌اند
- ✅ **مدیریت تصاویر**: امکان آپلود و مدیریت تصاویر برای تمام بخش‌ها
- ✅ **جستجو و فیلتر**: امکان جستجو و فیلتر در تمام صفحات لیست
- ✅ **مدیریت SEO**: تنظیمات SEO برای تمام صفحات

## نصب و راه‌اندازی

### پیش‌نیازها

- Python 3.8 یا بالاتر
- pip (مدیریت بسته‌های Python)

### مراحل نصب

1. **نصب وابستگی‌ها**:
```bash
pip install -r requirements.txt
```

2. **اجرای Migration‌ها**:
```bash
python manage.py makemigrations
python manage.py migrate
```

3. **ایجاد کاربر ادمین**:
```bash
python manage.py createsuperuser
```

4. **اجرای سرور توسعه**:
```bash
python manage.py runserver
```

5. **دسترسی به پنل ادمین**:
```
http://localhost:8000/admin/
```

## مدل‌های اصلی

### Core App
- `SiteSettings`: تنظیمات کلی سایت
- `ServiceCategory`: دسته‌بندی خدمات
- `Service`: خدمات
- `Testimonial`: نظرات و توصیه‌نامه‌ها
- `Goal`: اهداف کانون
- `HeroSlide`: اسلایدهای صفحه اصلی

### Events App
- `EventCategory`: دسته‌بندی رویدادها
- `Event`: رویدادها
- `EventImage`: تصاویر رویدادها
- `EventRegistration`: ثبت‌نام در رویدادها

### Tours App
- `TourCategory`: دسته‌بندی تورها
- `Tour`: تورها
- `TourImage`: تصاویر تورها
- `TourRegistration`: ثبت‌نام در تورها

### Blog App
- `BlogCategory`: دسته‌بندی پست‌ها
- `BlogPost`: پست‌های وبلاگ
- `BlogImage`: تصاویر پست‌ها
- `BlogComment`: نظرات پست‌ها

### Real Estate App
- `PropertyType`: نوع ملک
- `Property`: املاک
- `PropertyImage`: تصاویر املاک
- `PropertyInquiry`: استعلام ملک

### Contact App
- `ContactMessage`: پیام‌های تماس
- `ContactInfo`: اطلاعات تماس

## استفاده از پنل ادمین

پس از ورود به پنل ادمین، می‌توانید:

1. **تنظیمات سایت**: تنظیم اطلاعات کلی سایت، آدرس، تلفن، شبکه‌های اجتماعی
2. **مدیریت خدمات**: اضافه، ویرایش و حذف خدمات
3. **مدیریت رویدادها**: ایجاد رویداد جدید، مدیریت ثبت‌نام‌ها
4. **مدیریت تورها**: ایجاد تور جدید، مدیریت ثبت‌نام‌ها
5. **مدیریت وبلاگ**: نوشتن پست جدید، مدیریت نظرات
6. **مدیریت املاک**: اضافه کردن ملک جدید، مدیریت استعلام‌ها
7. **مدیریت نظرات**: تایید یا رد نظرات کاربران
8. **مدیریت پیام‌های تماس**: مشاهده و پاسخ به پیام‌های کاربران

## نکات مهم

- تمام محتوا از طریق پنل ادمین قابل ویرایش است
- تصاویر در پوشه `media/` ذخیره می‌شوند
- برای استفاده در production، باید `DEBUG = False` و `SECRET_KEY` را تغییر دهید
- توصیه می‌شود از PostgreSQL به جای SQLite برای production استفاده کنید
