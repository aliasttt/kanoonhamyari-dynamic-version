# راهنمای نصب و استفاده

## پیش‌نیازها

- Python 3.8 یا بالاتر
- pip (مدیریت بسته‌های Python)
- Virtual Environment (اختیاری اما توصیه می‌شود)

## مراحل نصب

### 1. ایجاد محیط مجازی (اختیاری اما توصیه می‌شود)

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 2. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### 3. اجرای Migration‌ها

```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. ایجاد کاربر ادمین

```bash
python manage.py createsuperuser
```

در این مرحله از شما نام کاربری، ایمیل و رمز عبور خواسته می‌شود.

### 5. جمع‌آوری فایل‌های استاتیک

```bash
python manage.py collectstatic --noinput
```

### 6. اجرای سرور توسعه

```bash
python manage.py runserver
```

سپس به آدرس `http://localhost:8000` بروید.

### 7. دسترسی به پنل ادمین

به آدرس `http://localhost:8000/admin/` بروید و با اطلاعات کاربری که ایجاد کردید وارد شوید.

## استفاده از پنل ادمین

### تنظیمات اولیه

1. **تنظیمات سایت**: 
   - به بخش "تنظیمات سایت" بروید
   - اطلاعات کلی سایت، آدرس، تلفن، شبکه‌های اجتماعی را وارد کنید
   - لوگو و فاویکون را آپلود کنید

2. **ایجاد خدمات**:
   - به بخش "خدمات" بروید
   - ابتدا یک "دسته‌بندی خدمات" ایجاد کنید
   - سپس "خدمات" جدید اضافه کنید

3. **ایجاد اسلایدهای صفحه اصلی**:
   - به بخش "اسلایدهای صفحه اصلی" بروید
   - اسلایدهای جدید اضافه کنید

4. **ایجاد اهداف**:
   - به بخش "اهداف" بروید
   - اهداف کانون را اضافه کنید

5. **ایجاد نظرات**:
   - به بخش "نظرات و توصیه‌ها" بروید
   - نظرات کاربران را اضافه کنید

### مدیریت محتوا

- **رویدادها**: ایجاد و مدیریت رویدادها
- **تورها**: ایجاد و مدیریت تورها
- **وبلاگ**: نوشتن پست‌های وبلاگ
- **املاک**: اضافه کردن و مدیریت املاک
- **پیام‌های تماس**: مشاهده و پاسخ به پیام‌های کاربران

## نکات مهم

1. **مدیریت تصاویر**: تمام تصاویر در پوشه `media/` ذخیره می‌شوند
2. **SEO**: برای هر صفحه می‌توانید تنظیمات SEO را در پنل ادمین وارد کنید
3. **نظرات**: نظرات کاربران باید از پنل ادمین تایید شوند تا نمایش داده شوند
4. **ثبت‌نام‌ها**: تمام ثبت‌نام‌های رویدادها و تورها در پنل ادمین قابل مشاهده هستند

## پشتیبانی

در صورت بروز مشکل، با تیم توسعه تماس بگیرید.
